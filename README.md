# Osumare Backend Assignment

This repository is the solution for the Osumare backend assignment (simple Express.js CRUD for tasks).

Clone the repository

```bash
git clone https://github.com/AdinathJabade/osumareBackendAssignment.git
cd osumareBackendAssignment
npm install
npm start
```

API base: http://localhost:8080/tasks
